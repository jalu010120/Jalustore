Pertanyaan yang sering diajukan

Berapa lama proses pengiriman Heart?
Karena aturan game Sky maksimal mengirim jumlah tertentu per hari, proses pengiriman bertahap (cicil). Kamu bisa cek progressnya di menu Tracking.
Apakah akun Sky saya aman?
100% Aman! JaluStore tidak pernah meminta email atau password akun kamu. Pengiriman heart murni menggunakan fitur in-game via Friend Link / QR Code.
Bagaimana sistem pembayarannya?
Pembayaran wajib dilunasi di awal sebelum proses pengiriman heart dilakukan. Kami menerima pembayaran via QRIS, DANA, GoPay, dan OVO.
Apakah butuh login akun?
Sama sekali tidak. Cukup masukkan Nickname dan Link QR/Sky Code kamu di form keranjang, lalu admin kami akan add friend kamu di dalam game.
